App para consultar los valores de las criptomonedas más importantes


Para la realización de esta aplicación haremos uso de una de las librerías más famosas de Python llamada Requests. Esta librería permite realizar peticiones HTTP, es decir que requests tienen una funcionalidad similar a AJAX de JavaScript lo que nos permitirá consultar la información de las criptomonedas enviando una petición a la API CoinCap API 2.0.


Lo primero que haremos será instalar Requests ejecutando el siguiente comando en consola y con nuestro entorno virtual activado:

pip install requests


Una vez instalada la librería ya estamos listos para empezar.

Debido a que estos componentes se crean a partir de una consulta a una API para obtener la información de las criptos, debemos tener en cuenta que estos componentes se crearán de forma dinámica.



El componente HomeScreen


La clase de HomeScreen será la pantalla donde se mostrarán nuestros componentes, una vez que se carga el componente podemos se ejecuta la función _load_cryptos(), en dicha función consultaremos las cryptos, la respuesta recibida la convertiremos a formato JSON y con un ciclo for generaremos los itesm para nuestra lista personalizada CryptoItemList.

Analizando el componente CryptoItemList


El el archivo py solo inicializaremos el componente y crearemos dos variables para colocar la cantidad de color verde si el crecimiento de la crypto ha sido positivo y rojo si su crecimiento ha sido negativo




En el archivo kv crearemos un GridLayout para colocar la información de las criptos de manera horizontal, notemos también que estamos utilizando un componente llamado AsyncImage que nos permitirá cargar los iconos de las imágenes de manera asíncrona sin interrumpir la carga del componente.





Botón para ocultar el listado de criptomonedas


Este botón será uno de los más interesantes del proyecto ya que aplicaremos algunas funcionalidades para poder deslizar el botón hacia abajo para ocultarlo y deslizar hacia arriba para mostrar el listado nuevamente, además de que agregaremos una animación para que sea más natural el movimiento.




Con la función _on_swipe() rastrearemos los movimientos que se hacen al mantener presionado el widget. _on_swipe() es una función nativa de la librería kivy y nosotros la utilizaremos para identificar si el componente se arrastra hacia abajo o hacia arriba y la altura del componente será reemplazada por la posición en el eje y en que estamos deslizando.




La siguiente función es on_state() otra función nativa de la librería para identificar el estado de un botón, si este se encuentra presionado o ha vuelto a su estado normal, nosotros la utilizaremos para saber si hemos dejado de deslizar el botón y saber en que posición se dejo de deslizar para determinar si cerraremos por completo el listado o lo abriremos.
